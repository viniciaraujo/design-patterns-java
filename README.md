# 🧩 Design Patterns em Java

## 📌 Sobre o projeto
Implementações dos principais padrões de projeto (Design Patterns) em Java,
desenvolvidas durante o curso de Engenharia de Software no IFG.
Cada pasta contém um padrão independente com seu próprio exemplo prático.

## 📁 Padrões implementados

| Padrão | Tipo | Descrição |
|---|---|---|
| **Singleton** | Criacional | Logger com instância única |
| **Prototype** | Criacional | Clonagem de documentos |
| **Builder** | Criacional | Construção de computadores |
| **Composite** | Estrutural | Hierarquia de funcionários e departamentos |
| **Adapter** | Estrutural | Adaptador de tomadas antiga/nova |

## 🛠️ Tecnologias
- Java
- IntelliJ IDEA
